Que es Meteor?
Una plataforma open-source  con un
conjunto de paquetes que proveen
características como reactividad,
sincronizacion de datos y
compensacion de latencia.
Con el objetivo de desarrollar de forma 
rápida y simple aplicaciones real-time
permitiendo maximizar la experiencia del 
usuario.

Instalación
  1. Necesitamos Node.js, npm y Mongo.
  2. curl https://install.meteor.com/ | sh
  3. npm install meteorite -g
Meteor y Meteorite 
No se maten con los nombres
  • Meteor usa un sistema propio de paquetes 
    desarrollados por ellos.

  • Meteorite extiende un poco a Meteor para
    permitir utilizar paquetes desarrollados
    por terceros.
Command line
  • $ meteor create myapp
  • $ meteor list
  • $ meteor add underscore
  • $ meteor remove underscore
  • $ meteor deploy myapp.meteor.com
  • $ meteor -h
$ meteor
Meteorite
  • $ mrt create myapp
  • $ mrt add router
  • $ mrt add router --pkg-version 0.3.4
  • $ mrt remove router
